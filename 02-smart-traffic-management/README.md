# Project 2: Smart City Traffic Management with Anomaly Detection

## 🚦 Overview
Industrial IoT (IIoT) traffic management system with distributed sensors, redundancy, and anomaly detection for a smart city intersection network.

## 🎯 Objectives
- Design scalable IIoT infrastructure
- Implement network redundancy and failover
- Configure wireless security for critical infrastructure
- Simulate DoS attacks and mitigation strategies

## 🛠️ Technologies Used
- **Platform:** Cisco Packet Tracer 8.2+
- **Protocols:** HSRP, VRRP, STP
- **Security:** WPA3 (simulated), VLANs, ACLs
