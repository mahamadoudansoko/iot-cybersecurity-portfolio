## 📚 Projects Overview

### [Project 1: Smart Building Security System with Intrusion Detection](01-smart-building-ids/)

**Status:** 🔨 In Progress
**Focus:** IoT security architecture, network segmentation, automated response

**Key Features:**
- 3-floor commercial building simulation in Cisco Packet Tracer 8.2+
- 5-VLAN network architecture (Management, IoT, Cameras, Users, Servers)
- 24 IoT devices: motion sensors, door sensors, smart locks, IP cameras
- Automated intrusion detection and lockdown response
- Centralized Syslog monitoring and security workstation

**Skills:** Network design · VLAN segmentation · Wireless security (WPA2) · ACLs · Port security · Automation
