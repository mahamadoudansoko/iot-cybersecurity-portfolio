# Project 3: Automated IoT Security Monitoring & Response System

## 🤖 Overview
Hybrid system combining Cisco Packet Tracer simulation with Python automation for SIEM-like security monitoring and incident response.

## 🎯 Objectives
- Integrate Packet Tracer with Python automation
- Implement network scanning and device discovery
- Build log parsing and anomaly detection
- Create security dashboard with visualization

## 🛠️ Technologies Used
- **Simulation:** Cisco Packet Tracer 8.2+
- **Programming:** Python 3.10+
- **Libraries:** scapy, pandas, matplotlib, flask
